DetectDerm is a mobile-based AI application that detects common skin diseases from photographs and provides medical advice in the Nepali language. It is designed specifically for users in Nepal — including those in rural and remote areas — where access to qualified dermatologists is extremely limited.
The application uses a Convolutional Neural Network (CNN) based on MobileNetV2 with Transfer Learning, trained on the publicly available HAM10000 dermatoscopic image dataset. DetectDerm works in both online mode (via a Flask REST API) and offline mode (using TFLite directly on the Android device), ensuring accessibility even without internet connectivity.
